FontAdapter: Tuning-free Font Customization in Visual Text Generation



Anonymous authors


TL;DR:  FontAdapter performs font customization in visual text generation, without fine-tuning.



Abstract

We present FontAdapter, a tuning-free framework that generates visual text in any unseen fonts in a few seconds, conditioned on a reference glyph image. Our approach is built upon the construction of synthetic datasets, leveraging large-scale online fonts and realistic background images generated by a model. We propose a two-stage training scheme tailored to the properties of our dataset: first learning font attributes from isolated glyphs, then fine-tuning on scene-text images to integrate these styles within diverse backgrounds. Our experiments show that FontAdapter achieves robust, high-quality customization across diverse fonts without additional fine-tuning. Moreover, we demonstrate various intriguing properties of FontAdapter, including cross-language font transfer, font style blending, and visual text editing.

Problem Statement

Our goal is to generate a visual text image with a customized font style, given a text prompt and a font reference image.

FontAdapter

Based on our synthetic datasets, we develop a two-stage training scheme that enables our model to effectively learn font attributes while optimally using our synthetic datasets.


In the first stage, FontAdapter is trained to generate a different text-only image with varied text content. This process allows FontAdapter to focus on extracting font attributes from glyph embeddings, concentrating on the finer details of the target glyphs.

In the second stage, FontAdapter is trained to generate a scene-text image that includes additional visual elements, such as background and glyph colors. This enables FontAdapter to flexibly customize font styles across diverse image contexts.
